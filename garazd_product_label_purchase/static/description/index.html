Print product barcode labels from purchase orders

Allows to print custom product barcode labels on different paper formats.

Module Features

Simply and fast utility to print product barcode labels
No configuration is needed

Usage

To print product labels from purchase orders open one or select a few from the purchases list:

  • Go to the Print menu and click on Custom Product Labels.

In the opened wizard form:

  1. Select the Label Template.
    2. Templates may have different designs, sizes, and paper formats. They can be added with our additional related modules.
  2. Change the number of labels that will be printed (if necessary).
    3. You can enter a quantity or use / buttons.
  3. If you need to set a specific quantity for each label, enter the value and click on Set quantity.
  4. After changing the label quantities you can restore the initial values. Click the button for this.
  5. Click on Preview or Print.
